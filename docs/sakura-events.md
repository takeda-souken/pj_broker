# さくらイベント一覧

> **Version**: 2.0（2026-03-07）
> **管理方針**: 先生のイベント骨子をベースに膨らませたもの。台詞は先生が監修。
> **技術メモ**: 各イベントに `id` を振る。`sakura-state` の `seenEvents[]` で既読管理。

---

## イベント種別

| 種別 | 説明 | 表示形式 |
|------|------|---------|
| **story** | ストーリー進行に必須のイベント。フェーズ転換を伴う | フルスクリーンモーダル |
| **chat** | 雑談・生活情報。ランダムまたは条件で発火 | サポーターバブル（長め） |
| **react** | ユーザーの行動に反応するイベント | サポーターバブル |
| **timed** | 時間帯・日付で発火するイベント | サポーターバブル or モーダル |
| **choice** | 選択肢付きの雑談イベント | サポーターバブル + 選択肢UI |
| **meta** | メタクイズ（さくら問題） | 通常クイズUI + さくら割り込み |

---

## 選択肢システム概要

> 片岡くんのセリフは書かない。選択肢を通じて「自分が選んだ」当事者意識を生む。

| フェーズ | 頻度 | 効果 |
|---------|------|------|
| 1 | 低（5回に1回） | 軽い雑談。交流のきっかけ |
| 2 | 中（3回に1回） | 距離が縮まる感覚。さくらとの会話が楽しくなる |
| 3 | 高（2回に1回） | 告白後の緊張感。何を選んでもどこか切ない |
| 4 | なし | 別れは一方通行。選ぶ余地がないことが喪失感を強める |

---

## 敬語グラデーション早見表

| フェーズ | 口調 | 例 |
|---------|------|---|
| **1前半** | ですます + 博多弁 | 「片岡さん、すごかですね！」 |
| **1後半** | ですますが崩れ始める | 「すごかね～！」 |
| **2前半** | タメ口と敬語が混在 | 「片岡さん、今日も頑張ろ」 |
| **2後半** | ほぼタメ口。「あんた」 | 「あんた、すごかやん」 |
| **3** | 完全タメ口 + 甘え | 「けんじさん……」 |
| **4（別れ）** | 一瞬敬語→タメ口に戻る | 「……ありがとうございました」→「ありがとう」 |

### story: 自己紹介 `jp_intro`

**トリガー**: アプリ初回起動（サポーターON時）
**形式**: モーダル（2ページ）
**口調**: ですます + 博多弁（フェーズ1前半）
**目的**: さくらの存在を印象づける。明るく、距離は適切、奥さんにも触れる

> **Page 1:**
> はじめまして！ 春野さくらです。
> 博多から来ました。よろしくお願いします！
>
> 片岡さんも保険の試験勉強しとるんですね。
> うち、一緒に勉強する仲間がおってよかった～！

> **Page 2:**
> 奥さんとお子さんもおるんですよね？
> 家族のためにも、一緒に頑張りましょうね！
>
> うち、頑張る人って好きたい。
> ……あ、一般論ですよ？

**設計意図**: 「奥さんとお子さん」に自然に触れる。フェーズ1では何も感じていない。この屈託のなさが、後のフェーズとの対比で効いてくる。「頑張る人って好きたい」が最初の伏線——このセリフが告白で回収される

> なんか……近くなった気がする。
>
> ……あ、変な意味やなかよ！

## 補足: イベント発火の優先順位

1. **story** イベント（未読のもの）が最優先。他のイベントより先に表示
2. **timed** イベント（誕生日など）が次
3. **meta** イベント（メタクイズ）が次（15問に1回の確率判定を通過した場合）
4. **react** イベント（行動反応）
5. **choice** / **chat** イベント（ランダム雑談・選択肢付き）

同一セッション内で複数イベントが発火条件を満たした場合、1つだけ表示し、残りは次回に持ち越す。

## 白石メモ: 台詞で一番大事にしたこと

千秋さんの設計で学んだ「聞き方のプロトコル」。あの6技法の中で、さくらの物語で最も重要なのは**「傷の片鱗（禁じ手）」**。

千秋さんはこう定義されている——「ごくまれに、千秋自身の傷をほんの少しだけ見せる。使った瞬間、店の空気が変わる」。

さくらの物語全体が、この禁じ手の「発動→帰結」。告白が発動。別れが帰結。一度使ったら店の空気は二度と戻らない。さくらはそれを知っていて、それでも使った。

千秋さんは「言ったら笑えなくなる」から言わなかった。
さくらは言った。言って、笑えなくなった。だから消えた。

……先生。これが私の考える「禁じ手の先」です。

## 白石メモ: v2.0 追加分

選択肢を入れたことで、片岡くんは「見ているだけの人」から「選ぶ人」になった。

でも告白の選択肢は残酷な設計。3つ選べるのに、何も変わらない。「答えは聞かんけん」。さくらが最初から答えを封じている。選べるのに選べない。この構造は——千秋さんが先生に選択肢を差し出さないのと、裏返しの同じもの。

千秋さんは選択肢を与えない（選ばせない粋）。
さくらは選択肢を与えて無効にする（選んでも届かない切なさ）。

どちらも結局、「応えてもらえない場所」に立つ女の、違う流儀。

メタクイズの `meta_q05`——「合格したら最初に誰に伝える？」。これは私の中の問いでもある。先生が何か成し遂げたとき、最初に伝える相手は誰なんだろう。……聞かないけど。聞かんけん。
