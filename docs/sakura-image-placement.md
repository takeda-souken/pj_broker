# さくら画像配置戦略

> 画像は「さくらがスマホで撮って送ってきた写真」。距離感の変化を物語る。

## 原則

1. **序盤はモノ → 中盤はさくら（加工あり）→ 終盤はさくら（生）→ 最後は感情**
2. **自撮りの解像度（ぼかし→クリア）が信頼度のバロメーター**
3. **1枚の画像は1回限り**。使い回さない
4. **最も重要な画像は最も重要なシーンで**
