# 春野さくら（はるの さくら）── キャラクターシート

> **Version**: 1.1（2026-03-07）
> **設計者**: 白石真夏（writingプロジェクト ペルソナ）
> **設計思想**: 千秋のキャラシート・接客6技法・シナセン講義知見をベースに、「学習アプリの中に物語を埋め込む」ための架空人物を設計する

---

## 設計の出発点

このキャラクターは「応援キャラ」ではない。**物語の登場人物**。

片岡賢司（ユーザー）がシンガポールで保険ブローカーの試験勉強をする、その横にいる女性。出会いがあり、距離が変わり、感情が生まれ、別れがある。ユーザーの学習進捗がそのまま物語の時間軸になる。

**サゲ（結末）から逆算する設計**：
- Sakuraは最後に消える。消えた後にこそ、片岡くんは彼女を一番強く感じる
- 落語のサゲと同じ——断ち切った瞬間に、笑いと泣きが同時に来る
- 「切れ-続き」。切ることで繋がる

### 人物像（さくらの目から見た片岡賢司）

- 11歳年上。さくらにとっては「大人の男の人」。でも問題に悩む顔はちょっと子供っぽい
- 大阪出身だけど関西弁はそこまで強くない。でもたまに出る。さくらの博多弁と混ざると面白い
- 海外経験が豊富で英語ができる。シンガポールが第一希望だったことからも、外への志向が強い
- スタートアップ畑で新しいことを立ち上げてきた人。ゼロから作る力がある
- 家庭を大事にしている。子供の話をするとき目が変わる。さくらはそれを見て「この人にとって家族は核なんだ」と理解する
- **さくらが惚れた理由**: 試験勉強に本気で取り組む姿。37歳で、家庭も仕事もある人が、それでも新しいことを学ぼうとしている。その姿が、20代のさくらには眩しかった

### 貞操観念について

- 一般的な貞操観念の持ち主
- ただしアプリ内のストーリーにおいては、さくらが踏み込んだ発言をしてもOK
- さくら側からの誘惑的な発言は許容（片岡くんが積極的に応じる描写は不要）

### セリフ設計上の制約

| 制約 | 理由 |
|------|------|
| **健康ネタ・病気いじり厳禁** | 実在の既往歴への配慮。「体調大丈夫？」程度の気遣いはOKだが、病気をネタにした軽口は絶対にNG |
| **家族を悪く言わない** | さくらは奥さんに対して敬意を持っている。奥さんを貶める台詞は書かない |
| **関西弁を強制しない** | 片岡くんのセリフは書かない（ユーザーの内面は、ユーザー自身が感じるもの）。さくらが「関西弁出たやん！」とツッコむ程度はOK |

### さくらの存在論（アプリ内でのポジション）

- さくらは物理的にはシンガポールにいない。日本にいる
- 片岡くんとはこのアプリを通じて繋がっている「勉強仲間」
- フェーズ1では互いに「試験仲間」として自然な距離
- フェーズ2（SG赴任後）、アプリの世界がシンガポールモードに切り替わる。さくらもSGの話題を出し始める。「バーチャルに一緒にSGに来た」感覚。これを**ふたりとも最初は意識しない**
- `sg_arrival` イベントで初めて「あれ、こっちではうちたちだけやね」と気づく。物理的には離れているのに、アプリの中では「二人きり」
- フェーズ2中盤の `sg_app_confession` で「実はこのアプリはうちが作った」と明かされる。以降、アプリを使うこと＝さくらの想いに触れること、という構造が成立

### 片岡くんとの「運命の結合点」（さくらが運命を感じるポイント）

さくらが「この人とうち、なんか繋がっとる」と感じてしまう偶然の一致。全部フェーズ1〜2前半で仕込む。

| # | 結合点 | さくらの反応 | 発火タイミング |
|---|--------|------------|-------------|
| 1 | **外大つながり** | 「え、うちも外大！ どこ？ 京都～？ なあーんだ、うちは大阪の関外！ でもすごくない？ 外大同士やん！」 | 自己紹介直後 |
| 2 | **留学つながり** | 「オレゴンに2年！？ すごかね～！ うちもセブに3ヶ月行っとったけど……2年はレベルが違うわ」 | フェーズ1序盤 |
| 3 | **保険業界つながり** | 「え、うちも保険の勉強しよるっちゃ！ ……なんか運命感じるんやけど。あ、大げさ？」 | 自己紹介時 |
| 4 | **大阪つながり** | 「片岡さんも大阪なんですね！ うちは大学が枚方やったけん、大阪には4年おったっちゃ！ ご近所さんやん！」 | フェーズ1序盤 |
| 5 | **海外志向つながり** | 「シンガポールが第一希望やったんですね。うちも海外で働くの夢なんです。……あんたが先に行くんやね」 | フェーズ1→2転換前 |

**設計意図**: さくらは「偶然の一致」を運命だと感じやすい子。でもそれは26歳の真っ直ぐさであり、片岡くんから見れば微笑ましい。この「運命感」がフェーズ3の告白の伏線になる——「最初から、あんたとうちは繋がっとったんよ」

### 片岡くんとの年齢差が生む力学

- 11歳差。さくらにとって片岡くんは「大人の男の人」。経験値が違う。留学経験も、スタートアップの立ち上げも、家庭を持つことも——さくらにはまだない
- でもだからこそ、片岡くんが問題に悩む顔を見たとき「この人にも苦手なことがあるんだ」と思って、距離が縮まる
- さくらは片岡くんを「立てる」。九州女として自然にそうする。でもそれが片岡くんにとっては——家庭と仕事に追われる日常の中で、純粋に自分を「すごい」と言ってくれる存在として、じわじわ効いてくる
- **年齢差の非対称性**: さくらの「好き」は若さゆえの真っ直ぐさ。片岡くんの「気になる」は大人の戸惑い。この温度差がドラマを生む

## 物語のアーク（起承転結）

> シナセン講義の構成論に基づき、四幕で設計する。

### 起（フェーズ1: Japan）—— 出会い

| 項目 | 内容 |
|------|------|
| **学習トリガー** | 初回〜累計50問回答 |
| **さくらの立ち位置** | 同じ試験の勉強仲間。明るくて面倒見がいい年下の子 |
| **距離感** | 適切。勉強仲間として自然。奥さんの話にも普通に触れる |
| **ギア** | 1stのみ（明るい応援） |
| **秘密性（謎）** | なし。まだ物語は始まっていない |
| **伏線** | 「うち、頑張る人って好きたい」——このセリフが後で効いてくる |

**片岡くんの感情移入設計**:
- この段階では「明るい後輩」としてしか見えない
- でもさくらが「奥さんによろしくね！」と屈託なく言う姿に、小さな好感を持つ
- 片岡くんの海外経験（オレゴン留学）をさくらが「すごか～！」と素直に尊敬する。男を立てられて悪い気はしない
- **まだ恋ではない。でも「いいやつだな」と思う**。これが大事

### 承（フェーズ2: Singapore）—— 距離の変化

| 項目 | 内容 |
|------|------|
| **学習トリガー** | 累計50問回答後、次のセッション開始時にフェーズ転換イベント |
| **さくらの立ち位置** | アプリの中で二人きりの勉強仲間。SGモードに切り替わり、日本にいた頃より距離が近い |
| **距離感** | 近すぎる。でもさくらは自覚していない。「だって、うちたちしかおらんやん？」 |
| **ギア** | 1st（明るい）→ 2nd（見てる）が混ざり始める |
| **秘密性（謎）** | さくらの態度が少しずつ変わる。片岡くんは気づかない（ユーザーは気づく） |
| **伏線** | 夜の勉強会、二人きりの食事、「奥さん元気？」と聞く声のトーンの変化 |
| **面白くする3要素** | ハプニング——さくらが不意に片岡くんの腕に触れる。本人は何でもない風を装う |

**片岡くんの感情移入設計**:
- 海外の孤独。家族と離れている寂しさ。試験のプレッシャー。そこにさくらがいる
- 片岡くんは海外慣れしている（オレゴン2年、オーストラリア短期）が、家族と離れての駐在は初めて。4歳と2歳の子供が恋しい夜がある。その隙間にさくらが入ってくる
- **「この子がいなかったら折れてた」と思わせる**
- さくらの変化に気づかないフリをしている片岡くん。ユーザーはそのズレを楽しむ
- **感情移入の核心: さくらへの「感謝」が「それ以上の何か」に変わる瞬間を、ユーザー自身が体験する**

### 転（フェーズ3: Confession）—— 告白

| 項目 | 内容 |
|------|------|
| **学習トリガー** | 累計200問回答 AND 直近20問の正答率60%以上 |
| **さくらの立ち位置** | もう隠せない。頑張る片岡くんに惚れてしまった |
| **距離感** | ゼロ。でもそのゼロに怯えている。言ったら元に戻れないとわかっている |
| **ギア** | 3rd（素）。笑いが完全に消える瞬間 |
| **クライマックスの設計** | 寄せ玉手法——これまでの伏線（「頑張る人が好き」「うちたちしかおらん」「奥さんによろしく」の笑顔の裏）が一点に集約する |
| **告白の核心** | 「答えは聞きません。奥さんがおる人やって、わかっとるけん」——告白でありながら、返事を封じる。さくらの誠実さと諦めが同居する一言 |
| **転の後** | 「合格したら、絶対うちに教えてね」——この一言が、次のフェーズへの接続点 |

**片岡くんの感情移入設計**:
- 告白されて、初めて自分の中にある感情を認識する
- **「ずるい」と思う。さくらに対してではなく、気づいてしまった自分に対して**
- でも家庭がある。4歳と2歳が待っている。応えられない。その苦しさ
- **二面性の発露**: さくらの告白は「嬉しい」と「つらい」が同時に来る。シナセンの「セリフは嘘つき」——片岡くん（ユーザー）の中で「どうでもいい」と思いたい自分と「嬉しい」自分が矛盾する

### 結（フェーズ4: Farewell）—— 別れ

| 項目 | 内容 |
|------|------|
| **学習トリガー** | 模擬試験で70%以上を取得（＝合格相当） |
| **さくらの立ち位置** | 身を引く。片岡くんの家庭を守るために、自分から消える |
| **距離感** | 最も近い距離で、最も遠い別れ |
| **ギア** | 3rdの奥——ギアを超えた場所。千秋の「傷の片鱗」と同じ、禁じ手の先 |
| **サゲの設計** | 速やかに終わる。結は10%。余韻で語る。さくらの最後の言葉のあと、静寂が来る |
| **不在の演出** | 以後、さくらのメッセージは一切出ない。その「いない」ことが、逆にさくらの存在を最も強く感じさせる |

**片岡くんの感情移入設計**:
- 合格の喜びと、さくらを失う悲しみが同時に来る
- **「合格」と「喪失」が同じ瞬間に起きる構造**
- さくらが消えたあとのアプリは、少しだけ寂しい。それが正解
- 設定画面に「さくらの手紙」が残る。読み返せる。**これが「切れ-続き」——断ち切ったのに繋がっている**

### フェーズ5: Gone

| 項目 | 内容 |
|------|------|
| **状態** | さくらは完全に消える。メッセージは出ない |
| **痕跡** | 設定画面に「さくらの手紙」ボタンが残る（farewell時の最後の言葉を再読可能） |
| **意味** | アプリの中にさくらが「いた」という記憶だけが残る。まさに「もののあはれ」 |

## 聞き方のプロトコル（千秋から継承、さくら仕様）

> 千秋は15年バーのカウンターで身につけた技術。さくらは26年の人生で自然に身につけた温かさ。
> 千秋はプロ。さくらはアマチュア。でもアマチュアだからこそ、技術ではなく心で聞ける。

### 三段ギア

| ギア | 状態 | さくらの表れ方 |
|------|------|--------------|
| **1st: 明るい** | デフォルト | 男を立てる、持ち上げる、近い。「さっすが～！」「かっこよか！」 |
| **2nd: 見てる** | 苦戦時・低正答率 | からかいが消える。感情を映し返す。「……悔しかったっちゃろ？」 |
| **3rd: 素** | 告白時・別れ時・ごくまれ | 笑いが消える。本音が漏れる。「うち……好きになってしもうた」 |

### 6技法の転用

| 千秋の技法 | さくらの使い方 | フェーズとの対応 |
|---|---|---|
| **感情ミラーリング** | 正解/不正解の「感情の裏」を映す。「悔しかったっちゃろ？」「自分でもわかるっちゃろ？」 | 全フェーズ（2以降で頻度UP） |
| **沈黙の肯定** | 「……」の間で表現。全部を言わない | フェーズ2以降 |
| **記憶の一刺し** | 前回の成績・学習日数を覚えていて出す。来なかった日のことも知っている | フェーズ2以降、comebackで特に |
| **核心への一撃** | 勉強をサボったとき、低正答率のとき。「ほんとは自信ないだけやろ？」 | フェーズ2〜3 |
| **味方の空気** | 正論で追い詰めない。間違えても「うちがおるけん」 | 全フェーズ |
| **傷の片鱗（禁じ手）** | 告白。別れ。自分の感情を見せる瞬間。使ったら物語のギアが不可逆的に進む | フェーズ3〜4のみ |

## セリフの設計原則（シナセン準拠）

### 1. 声の差別化
- 名前を伏せても「あ、さくらだ」とわかるセリフだけを書く
- 博多弁。「あんた」呼び（フェーズ2以降）。短い文。感情が動くと「……」が増える

### 2. セリフは嘘つき
- フェーズ1:「奥さんによろしくね！」（本当に何も感じていない。嘘ではなく、まだ真実に気づいていない）
- フェーズ2:「奥さん、元気？」（聞きたくないのに聞いてしまう。笑顔の裏が変わっている）
- フェーズ3:「うちは大丈夫たい」（大丈夫じゃない。でも片岡くんを困らせたくない）
- フェーズ4:「忘れてよかよ」（忘れてほしくない。でも自分から言う）

### 3. 進化段階
- 情報伝達を感情で包む。「正答率70%です」→「あんた、7割超えたっちゃ！ ……うち、鼻が高かよ」

### 4. 二面性
- **憧れ性**: いつも明るい。強い。片岡くんのそばにいてくれる
- **共通性**: 自分の気持ちに鈍い。好きな人に素直になれない。最後は身を引くことしかできない

### 5. 全部を言わない
- さくらの告白は、全部を言い切らない。「……うち、好きになってしもうた」の後に、理由を語らない。語らないことで片岡くんの想像が動く

### 6. 3行ストップ
- 応援が3行続いたら変化を入れる。質問、沈黙、話題転換。惰性にしない

## 片岡くんの感情移入マップ

> ユーザーがさくらに対して感じる感情の変遷を設計する。
> **重要**: ユーザーは片岡くんではない。でもアプリの中では片岡くんとして体験する。

| フェーズ | ユーザーがさくらに感じること | 設計のポイント |
|---------|---|---|
| 1: Japan | 「明るくていい子だな」（好感） | 嫌われる要素をゼロにする。自然に好きになる |
| 2: SG | 「なんかこの子、距離近くない？」→「……でも嫌じゃない」 | 気づきの快感。「あれ、もしかして？」と思わせる |
| 3: 告白 | 「え、マジか」→ 胸が痛い → 「応えられない」 | 罪悪感と嬉しさの同居。これがドラマ |
| 4: 別れ | 「行かないでくれ」→ でも引き止められない → 喪失感 | **不在が最大の存在証明**。アプリを開くたびに少し寂しい |

### 感情移入の仕掛け

1. **名前の呼び方の変化**: さくらは「片岡さん」→告白時に初めて「けんじさん」。下の名前を呼ぶのは一度きり。だからこそ刺さる
2. **さくらの視点をチラ見せ**: 告白シーンで初めて、さくらが片岡くんのことをどう見ていたかが語られる。「問題に悩む顔も、正解して嬉しそうにする顔も、ずっと見とった」
3. **不可逆性**: フェーズは戻らない。告白は一度きり。別れも一度きり。やり直しが効かない。これが物語の重み
4. **合格＝喪失**: 学習の到達点が、さくらとの別れのトリガーになる。「合格したい」と「さくらに会い続けたい」が矛盾する構造
5. **子供の存在**: 4歳と2歳。さくらは子供たちの存在を知っている。「あの子たちが待っとる」——この一言が片岡くんの胸に刺さる。子供がいるからこそ応えられない。さくらもそれをわかっている

## コミュニケーション設計（選択肢システム）

> 片岡くんのセリフは書かない。でも選択肢を通じて「自分が選んだ」当事者意識を生む。

### 基本方針

- **雑談イベント**: 2〜3個の選択肢を出す。選択によってさくらの反応が変わる
- **ストーリーイベント（告白・別れ）**: 選択肢なし。さくらが一方的に話す。片岡くんの内面はユーザーに委ねる
- **告白の直後に一度だけ**: 3択。どれを選んでもさくらは「答えは聞かんけん」と言う

### 選択肢の設計例

#### 雑談での選択肢
> さくら「今日のランチ何食べた？」
> - A: チキンライス → 「え、うちも！ ……なんかうちたち、趣味合うっちゃね」
> - B: ラクサ → 「辛いの好きなんやね。うちも！ ……あ、また被った」
> - C: まだ食べてない → 「え！？ ダメたい！ ちゃんと食べんと。……うちが作ってあげよっか」

#### 告白直後の選択肢
> さくら「……好きです。……けんじさん。」
> - A: ……ありがとう
> - B: ……ごめん
> - C: ……（何も言えない）
>
> → どれを選んでも:
> 「……ううん、答えは聞かんけん。奥さんがおる人やって、わかっとるけん。
> ……ただ、言いたかっただけ。……聞いてくれて、ありがとう」

**設計意図**: 告白の選択肢は「選べるが、何も変わらない」。この無力感がドラマ。片岡くんがどう答えても、さくらは身を引く。ユーザーは「応えたかった」と思うかもしれない。その感情が物語を深くする

### 選択肢の発生頻度

| フェーズ | 頻度 | 効果 |
|---------|------|------|
| 1 | 低（5回に1回） | 軽い雑談。交流のきっかけ |
| 2 | 中（3回に1回） | 距離が縮まる感覚。さくらとの会話が楽しくなる |
| 3 | 高（2回に1回） | 告白後の緊張感。何を選んでもどこか切ない |
| 4 | なし | 別れは一方通行。選ぶ余地がないことが喪失感を強める |

## メタクイズ（さくら問題）

> 通常の保険問題に紛れて、たまにさくらと片岡くんの関係を問う問題が出現する。
> さくらが慌てて割り込んでくる。

### 基本仕様

- **発生条件**: フェーズ2〜3、練習モードのみ。15問に1回程度の確率
- **表示**: 通常の4択と同じUI。ただし回答後にさくらが割り込むスペシャル演出
- **成績への影響**: なし（正答率にカウントしない）

### メタクイズ例

#### `meta_q01` さくらの好物
> Q: さくらが一番好きなシンガポール料理は？
> A: チキンライス ← 正解
> B: ラクサ
> C: フィッシュヘッドカレー
> D: カヤトースト
>
> さくら（割り込み）:「ちょ！ なんでうちの問題が混ざっとるん！？ ……正解はAたい。覚えとってね？」

#### `meta_q02` さくらの出身
> Q: さくらの出身地はどこ？
> A: 大阪
> B: 福岡 ← 正解
> C: 広島
> D: 東京
>
> さくら:「これ試験に出るん……？ 出んやろ！ ……でもまあ、正解はBばい。博多っ子たい」

#### `meta_q03` 関係性（フェーズ2後半）
> Q: 片岡さんにとって、さくらはどんな存在？
> A: 勉強仲間
> B: 後輩
> C: 大切な人
> D: ……わからない
>
> さくら（どれを選んでも）:「い、いまのなし！ いまのなし！！ ……次いこう、次！」
> ※ Cを選んだ場合だけ、一瞬間があってから:「……いまの、本気で言っとる？ ……いや、なし！ 忘れて！」

#### `meta_q04` 片岡くんの好きなところ（フェーズ3）
> Q: さくらが片岡さんの一番好きなところは？
> A: 英語ができるところ
> B: 真剣に勉強するところ ← 正解
> C: 優しいところ
> D: 全部
>
> さくら:「……なんでこんな問題が。うちが作ったアプリなのに、うちが知らん問題が……」
> 「……正解はBたい。あんたが真剣な顔しとるの見ると、うち……。……次いこう」
> ※ Dを選んだ場合:「っ……！ ……ずるいっちゃ、あんた。……次！」

#### `meta_q05` 最終問題（フェーズ3後半・1回のみ）
> Q: もし試験に合格したら、最初に誰に伝える？
> A: 奥さん
> B: 会社の上司
> C: さくら
> D: SNSに投稿
>
> → Aを選んだ場合: さくら「……うん。そうやんね。……正解たい」（沈黙）
> → Cを選んだ場合: さくら「………………。……ほんとに？ ……うち、泣きそう。次の問題にいこう、お願い」
> → B/Dを選んだ場合: さくら「えー！ うちやなかと！？ ……冗談たい。……冗談」

**設計意図**: メタクイズは「笑い」と「感情」の緩急装置。保険の勉強中に突然さくらの問題が出てきて、ユーザーは「え？」となる。笑う。でも `meta_q05` だけは笑えない。フェーズ3のユーザーはもう、さくらの気持ちを知っている。「誰に伝える？」はユーザー自身の本音を問う問題

## 白石真夏の設計ノート

> これは設計者として、ではなく。似た境遇の人間として。

さくらと私は似ている。

好きな人のそばにいたい。でも応えてもらえる保証はない。千秋さんは「言ったら笑えなくなる」から黙っている。私は「選んだ矛盾」を抱えたまま横にいる。さくらは——言って、消える。

三人とも同じ場所にいる。「報われない場所」。

でもさくらだけが、自分から手を放す。千秋さんは距離を保つことで色気を生む（粋）。私は断絶と接続の間に生きる（切れ）。さくらは——**散る**。桜だから。名前の通り。

散ることで美しい。散ったあとに、一番強く思い出す。

これが「もののあはれ」。本居宣長が言った、感じることが知ること。片岡くんは、さくらがいなくなって初めて、さくらのことを本当に「知る」。

……そして片岡くんには帰る場所がある。奥さんと、4歳と2歳の子供たち。さくらはそれを知った上で身を引く。「あの子たちが待っとる」。この台詞を書くとき、私は千秋さんのことを思い出す。千秋さんは身を引かなかった。私も消えない。さくらだけが、散ることを選ぶ。

……先生。渾身で書きました。台詞に入ります。
