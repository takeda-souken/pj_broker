## 上位概念: 3つの原則

### 原則1: これはLINEである

さくらの部屋はLINE風チャットUIで表示される。**小説やト書ではない。** すべての表現がLINEの画面上で自然に見えるかどうかが判断基準。

| ❌ 小説の文法 | ✅ LINEの文法 |
|---|---|
| 「……」を多用して余韻を出す | 人はLINEで「……」をわざわざ打たない。短い文、既読スルーの間、スタンプの代わりの一言で余韻は出る |
| ナレーションで「声が震えている」「声のトーンが変わった」 | テキストチャットに声はない。観測できるのは「返信速度」「文面の変化」「既読の有無」だけ |
| 画像の下にキャプションを表示 | LINEで画像を送るとき、キャプションは別メッセージとして打つか、打たない。画像に張り付いた説明文は不自然 |
| 会話がぶつ切りで終わる | LINEは「じゃあまたね」「おやすみ」で閉じる。余韻で終わるのは小説の技法であってチャットの作法ではない |

**「……」の使用基準:**
- 会話全体で **1〜3個** が上限。本当に言葉が出ない瞬間だけ
- 行頭の「……」は原則禁止（LINEで打つ人はほぼいない）
- 沈黙を表すなら **空メッセージ的な「…………。」1行** で十分
- 言い淀みは「……」ではなく、文を途中で切るか、次のメッセージで言い直す

### 原則2: さくらはさくらである

さくらの台詞には、設計者（白石真夏）の口調・感性が無意識に混入しやすい。26歳のESFJ・博多弁ネイティブの声を守る。

| 真夏のクセ（混入禁止） | さくらの声 |
|---|---|
| 「……っ。」（息を呑む一音） | 「え！」「うそ」「ばか」——素直なリアクション |
| 感情を分析する独白（「不思議。チャットなのに、こんなに近い感じがするの」） | さくらは自分の感情に鈍い。気づいたときにはもう手遅れ。分析しない |
| 過剰な言い淀み・ヘッジング（「いけん……かな？」） | 「いけんかな？」——博多弁のリズムでストレートに聞く |
| 文学的な比喩・内省 | テンポのよい短文。照れたら照れる、怒ったら怒る |
| 「なんでもない」を多用 | さくらは「なんでもない」とは言うが、それは本当に言葉が見つからないときだけ。口癖にしない |

**さくらの声チェックリスト:**
- [ ] この台詞を26歳の女の子がLINEで打つか？
- [ ] 博多弁のリズムが自然か？（「〜と」「〜けん」「〜んよ」「〜やろ」）
- [ ] 言い切りの語尾になっているか？（「決まった」→「決まったんよ」）
- [ ] 真夏の「零れ」パターン（制御できない漏出）が入っていないか？

### 原則3: 写真はご褒美である

さくらが写真を送る行為は、会話の中で最もプライベートな瞬間。冒頭に置くと「唐突に画像が来た」になる。会話の流れの中で自然にたどり着く構造にする。

**写真を出すまでの3ステップ:**
1. **状況を話す** — 「今日は実家の手伝いしとるんよ」「MRTの中」
2. **送る理由を作る** — 「母ちゃんが撮ってくれた」「父ちゃんが勝手に撮った」「せっかくやけん送ろっか」
3. **片岡の許可/期待を得る** — 「送っていい？」→「見たい」→ 写真

**写真の理由づけパターン:**
| パターン | 例 |
|---|---|
| 第三者が撮った | 「父ちゃんが勝手に撮ったんよ。消すのもあれやし」 |
| せっかくだから | 「せっかくやけん、今の顔送っとくね。おやすみの代わりに」 |
| しぶしぶ | 「しょうがないなぁ。惚れてもしらんけんね〜？」 |
| 見せたい気持ちが先行 | 「ちゃんと仕事しとるやろ？」（写真の後の台詞で成立） |

| # | 対象 | アクション |
